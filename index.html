Bài 1: Tính tiền lương nhân viên

Sơ đồ 3 khối

INPUT PROGRESS OUTPUT
  • Lương 1 ngày : ?;
  • Số ngày làm : ?;
  • DOM tới button để lấy sự kiện onclick;
  • Tạo biến lương 1 ngày: luongMotNgay;
  • Tạo biến số ngày làm: soNgayLam;
  • Tạo biến tiền lương: tienLuong;
  • Tạo biến total để tính lương;
  • Tính lương: total = luongMotNgay*soNgayLam;
  • Gán giá trị total vào thẻ HTML với id="tienLuong";
  • Show kết quả tiền lương;

Bài 2: Tính giá trị trung bình

Sơ đồ 3 khối

INPUT PROGRESS OUTPUT
  • 5 số thực được nhập từ người dùng;
  • DOM tới button để lấy sự kiện onclick;
  • Tạo biến và lấy giá trị 5 số thực: n1, n2, n3, n4, n5;
  • Tạo biến giá trị trung bình: trungBinh;
  • Tạo biến kết quả: ketQuaTB;
  • Tính trung bình: trungBinh = (n1 + n2 + n3 + n4 + n5) / 5 ;
  • Gán giá trị vào thẻ HTML với id="ketQuaTB";
  • Show giá trị trung bình;
